Project Overview

The EduSphere is a terminal-based educational management system built entirely in C. It provides an intuitive way for managing students, courses, and faculty members, all within a simple command-line interface. This system offers role-based access to ensure that users (admins, faculty, and students) have the appropriate permissions.

With features like user authentication, course tracking, and file management, EduSphere makes education management streamlined and efficient. It was developed with a focus on providing core functionality for educational institutions.

Tech Stack

Programming Language:

(Terminal-based)

Key Features

  • Authentication System: Role-based authentication allows for different access levels (Admin, Faculty, Student).
  • Course Management: Admins can add, update, or delete courses, and students can enroll in them.
  • Student Tracking: The system tracks student grades and attendance, providing a clear overview of their performance.
  • File Handling: The system allows for storing and retrieving student and course-related files, such as assignments and grades.

Challenges Faced

  • Managing File Handling in C: Since C lacks built-in support for file management, a lot of custom code had to be written to store and retrieve files.
  • Implementing Role-Based Access: Ensuring that users had the right permissions based on their role was complex, but was solved by designing an efficient user authentication system.
  • Complexity of the Command-Line Interface: Ensuring the program was user-friendly despite being terminal-based required clear instructions and concise error handling.

Future Plans

While EduSphere is a terminal-based solution, future plans include:

  • Developing a graphical user interface (GUI) for more ease of use.
  • Implementing a cloud-based storage solution for data persistence.
  • Integrating real-time communication for notifications between faculty and students.
  • Adding advanced student analytics features for better insights on student progress.
